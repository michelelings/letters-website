# Letters SEO Architecture & Topical Map

This document outlines a scalable SEO architecture for the Letters
language learning platform.\
The structure supports: - Multiple languages - Translation intent -
Learning intent - Localized URLs - Programmatic SEO - Strong hub pages
and internal linking

------------------------------------------------------------------------

# 1. Top Level Structure

The site consists of three main SEO systems:

/learn\
/translate\
/languages

Purpose:

-   **Learn** тЖТ people who want to learn a language\
-   **Translate** тЖТ people who want to translate words\
-   **Languages** тЖТ topical authority hubs

# 10. Internal Linking Model

Each page links to three hub types.

Word pages link to:

-   topic page
-   translation page
-   language hub

Topic pages link to:

-   word pages
-   phrase pages
-   travel pages

Translation pages link to:

-   learning pages
-   topic pages

This creates a dense internal knowledge graph.

# Key Insight

Translation pages likely generate the largest traffic volume because
users search queries like:

-   how to say X in Spanish
-   how to say X in French
-   what does X mean in English

Learning hubs build topical authority so search engines trust the site.

Both layers are essential.
